Myna Project

Myna is an energy management application. This project was thought to give a simple user-friendly interface to both manage the heating system and monitor climatic variables in a small-medium house.

Its flat and minimal style allows the app to be used on all devices and in a simple way even by people not really technology-friendly.

Main features are:

  • temperature control using manual, automatic threshold or time programming;
  • real-time monitoring;
  • data storage and graphic representation up to one-year backward;
  • heat countering with economic values estimation;
  • alerting system (email and/or SMS);
  • remote exclusive access from any device connected on the web.

The application could run on a single board computer connected wired or wireless to sensors and actuators easily to configure.
